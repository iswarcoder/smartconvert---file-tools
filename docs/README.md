# ⚡ SmartConvert – AI Powered PDF Toolkit

🔗 **Live Demo:** https://smartconvert-frontend.onrender.com/

---

## 📌 Overview

**SmartConvert** is a web-based PDF toolkit that allows users to easily convert, merge, split, compress, and edit PDF files. It also integrates AI-powered features like summarization and translation for smarter document handling.

---

## 🚀 Features

* 📄 **Convert File** – Convert office documents into PDF
* 📎 **Merge PDF** – Combine multiple PDFs into one
* ✂️ **Split PDF** – Extract selected pages
* 📦 **Compress PDF** – Reduce file size
* 🖼️ **Image to PDF** – Convert images into PDF
* ✏️ **Edit PDF** – Basic editing (extendable)

### 🤖 AI Features (Upcoming / Integrated)

* 🧠 AI PDF Summarization
* 🌐 AI Translation (Multi-language support)
* 💬 Chat with PDF (future scope)

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Libraries & Tools

* pdf-parse (PDF text extraction)
* Multer (file upload handling)
* Gemini AI API (AI processing)

---

## ⚙️ How It Works

1. User uploads a file (PDF/Image)
2. Backend processes file using PDF libraries
3. AI (Gemini API) handles:

   * Summarization
   * Translation
4. Processed result is displayed or downloaded
